两数之和

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

可以按任意顺序返回答案。

思路：首先创建数组result，用来存放找到的两个整数，然后使用for循环依次遍历整个数组里面的两个组合。如果有某两项组合的和等于她target，则把这两个数放进result这个数组，把returnSize设为2，最终返回result数组。倘若没有两个组合的和等于target，则把returnSize设为0，返回NULL。

回文数

给一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

例如，121 是回文，而 123 不是。

思路：回文数不可能是负数。首先定义一个int型original来存放最初的x，再定义reversed来存放反转后的数字，之后定义remainder来计算每一位的数字，在while循环中，先从个位开始计算每一位，再让reversed乘10加上这次得到的这个位的数，最后x除10；最终得到反转后的数字。最后判断两者是否相等即可

最长公共字符

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 “”。

思路：倘若字符串的数量为空，就返回NULL。倘若字符串只有一个，就返回这个字符串。拿第一个字符串进行比对，因为最长公共字符串的长度不可能超过第一个的长度。逐一比对第1个字符串的第i位和第j个字符串的第i位。如果两个不相等或者第j个字符串走到尽头了，创建字符串，将第1个字符串的前i位赋值给结果字符串。如果遍历完，所有字符串都相同，则返回第一个字符串。

删除有序数组中的重复项

给一个非严格递增排列的数组 nums ，请原地删除重复出现的元素，使每个元素只出现一次，返回删除后数组的新长度。元素的相对顺序 应该保持一致 。然后返回nums中唯一元素的个数。

思路：由于数组是非严格递增数组，所有重复元素的出现都是连续的。故使用memmove来实现。遇见重复元素，就把第二个重复元素和后面的部分全部往前移动一位，覆盖掉第一个重复的元素。

移除元素

给一个数组 nums 和一个值 val，需要原地移除所有数值等于 val 的元素。元素的顺序可能发生改变。然后返回 nums 中与 val 不同的元素的数量。

思路：使用双指针法，使用快指针遍历整个数组，遇见和该值不相等的元素，就将其赋值给nums[slow]。slow的值就是nums中与val不同的元素的数量

删除链表倒数第n个节点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。如List=[1,2,3,4,5],n=2,删除后List=[1,2,3,5]

解

先添加一个虚拟头节点，再使用双指针法，让fast指针先走n+1步，然后再让fast和slow同时移动，当fast走到NULL时，slow就是待删除节点的前驱节点

因为我们永远从“前一个节点”去删除，不会出现访问空指针。
即使要删除头节点（如倒数第 N 个正好是第一个），虚拟头节点也能安全处理。
所有边界情况（链表长度 = 1、n = 链表长度等）都能覆盖。
